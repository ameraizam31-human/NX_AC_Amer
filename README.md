# 🚀 NX_AC_Amer: Streamlit Data Applications

Welcome to the **NX_AC_Amer** repository! This workspace contains professional-grade, interactive web applications built using Python and Streamlit, designed to automate Data Exploration and Data Cleaning processes.

**Developer:** Amer Aizam 

---

## 📂 Projects Included

### 1. Complete EDA Application (`Eda_Amer.py`)
An automated Exploratory Data Analysis (EDA) tool that allows users to upload datasets and instantly generate insights without writing a single line of code.
* **Features:**
  * 📁 Supports both CSV and Excel file uploads.
  * 📊 Automatic summary statistics for numerical and categorical features.
  * 📈 Interactive visualizations (Correlation Heatmaps, Scatter Plots, etc.).

### 2. I SETEL - Data Cleaner Pro (`data_cleaner_amer.py`)
A robust data preprocessing application to clean messy datasets with a user-friendly interface.
* **Features:**
  * 🔍 Initial Data Diagnostics (Instantly detects missing values and duplicates).
  * 🗑️ One-click removal of missing values and duplicate records.
  * 🛠️ Smart handling of missing values (e.g., mean imputation).
  * 💾 Export functionality to download the cleaned dataset instantly.
